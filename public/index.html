How This Tracker Works

1
Stripe captures payer details
Parent pays via Stripe. Stripe records name, email, amount and date — but NOT the student name.
2
Import the Stripe CSV
Export from Stripe Dashboard → Payments → Export CSV. Import here via the Payments tab. All payer data populates automatically.
3
Teacher assigns student names
Share this URL with your school contact (Teacher PIN). They use the Match Students tab to click each row and type the student name. Saves instantly to the server.
4
Export the complete record
Once all students are matched, export the full CSV from the Payments tab for your school records.

Render Environment Variables

Set these in your Render Dashboard → Service → Environment to customise PINs and data location:

ADMIN_PIN=your-admin-pin
TEACHER_PIN=your-teacher-pin
DATA_DIR=/var/data # set to your persistent disk mount path
⚡ Persistent Disk: Add a Render Disk mounted at /var/data and set DATA_DIR=/var/data. Without a disk, data resets on each deploy — the disk costs ~$0.25/GB/month on paid plans.
